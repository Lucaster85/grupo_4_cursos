Curso de Diseño Web: HTML y CSS desde cero hasta avanzado.

El curso se divide en varias secciones, en las cúales podrás aprender los fundamentos de estas dos tecnologías. A continuación puedes ver una breve descripción de los contenidos de cada bloque:

Introducción

Aquí veremos la presentación del curso, y luego procederemos a descargar e instalar en nuestro computador las herramientas necesarias para desarrollar un proyecto web, por ejemplo, navegadores y editores de códigos.

HTML

En esta sección entraremos en materia y conoceremos en profundidad todo lo relacionado al lenguaje HTML.

CSS

Este es un bloque extenso en el que aprenderemos a utilizar hojas de estilos para definir el aspecto visual de los contenidos de una página web.

Proyectos finales

Aquí pondremos en práctica todo lo aprendido en los bloques anteriores, y en ese sentido diseñaremos 3 proyectos web del mundo real:

  1. Crearemos un landing page o página de aterrizaje para una empresa que imparte un curso de marketing digital.
  2. Después, diseñaremos un sitio multipáginas para una oficina de abogados.
  3. Por último, crearemos un one page site o sitio web de una sóla página para una empresa de servicios de catering.
